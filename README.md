AL-Manhaj is a simple, clean web platform for Islamic content in Arabic. It brings together key features such as Quran access, Hadith, daily remembrance, electronic tasbih, podcasts, lectures, and prayer times—aimed at helping users stay connected to their faith in a structured and accessible way.

You can access the live website here:  
https://manahij.vercel.app/

---

## Features

- **Home / Navigation** across site sections
- **Quran** browsing + reading experience
- **Hadith** collection
- **Electronic Tasbih** (مسبحة إلكترونية) with interactive counting
- **Daily Remembrance & Adhkar** (أذكار اليوم) / morning & evening collections
- **Prayer Times** (أوقات الصلاة) with location selection
- **Self-accounting prompts** (محاسبة النفس)
- **Podcasts & Lectures** sections for Islamic audio/content
- **Radio section** for Quran streaming (إذاعة القرآن الكريم)
